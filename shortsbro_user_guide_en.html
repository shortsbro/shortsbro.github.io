YouTube Controls (Watch/Shorts)

Floating controls appear only on YouTube Watch or Shorts pages.

Floating Control Box (Draggable)

Media controls are shown as a floating button cluster near the bottom of the screen. Long-press and drag the cluster to reposition it.

Tap the hamburger button (menu) to open Settings. Layout (1/2 columns) and the volume slider are configured in Media Controls.

Seek Buttons: Short / Long Press

Short press replay_5/forward_5 to jump 5 seconds. Long press to select a 10-second jump.

Speed & Quality (PRO)

In the PRO build, tapping the speed button opens a sheet where you can select both playback speed and video quality on the same screen. Available quality options depend on the YouTube player (e.g., Auto, 720p, 1080p). In the general build, speed/quality UI may be limited.

Shorts Auto-Next

When Auto Next for Shorts is enabled, ShortsBro automatically advances to the next Short after the current one ends.

Behavior

1) Detects Shorts completion
2) Attempts to move to the next Short
3) Uses fallback logic if looping/replay is detected

Delay

Configure the delay in Settings: 0s, 0.5s, or 1.0s.

Note

Auto-next depends on YouTube DOM/player structure and may break when YouTube updates.

Settings

Media Controls (Floating UI)

Volume Slider: show/hide the vertical volume slider next to the floating controls.
1 Column / 2 Columns: choose one or two columns for the floating buttons.
Toggle buttons individually: Sleep Timer Button, Speed Button, Skip Back Button, Skip Forward Button, Back Button.

Note: the timer icon represents Sleep Timer (PRO feature), not playback speed.

Controls Background Opacity (Slider)

Adjust the opacity of the background scrim behind the floating control box. Higher values improve visibility; lower values reduce screen obstruction.

PRO Differences (Reference)

PRO unlocks features such as Sleep Timer and the speed/quality selection sheet. Labels and available toggles may differ by edition.

Debug

Useful when reporting issues or collecting evidence for reproducible bugs.

Debug Overlay

Shows URL, load state, errors, and console logs on-screen for screenshots.

Debug Log File

Writes debug events to a text file and keeps only the most recent ~30 minutes. Use Clear Log to reset.
